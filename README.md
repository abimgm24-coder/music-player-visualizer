# Web Music Player with Audio Visualizer

## Features
- Play / Pause / Next / Previous
- Progress bar with seek
- Playlist with clickable songs
- Current song highlight
- Auto-next track
- Volume control
- Current time / duration display
- Real-time audio visualizer using Web Audio API
- Responsive layout for mobile and desktop

## Technologies
- HTML / CSS / JavaScript
- Web Audio API & Canvas
